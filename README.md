## 2. Cấu trúc thư mục
Dự án được chia thành 3 phần chính theo yêu cầu của đồ án:
* `part1/`: Các phép toán ma trận cơ bản (Khử Gauss, Định thức, Nghịch đảo, Hạng và Cơ sở).
* `part2/`: Phân rã ma trận (SVD, Chéo hóa) và Trực quan hóa biến đổi tuyến tính bằng Manim.
* `part3/`: Giải hệ phương trình tuyến tính, so sánh hiệu năng (Benchmark) và phân tích tính ổn định số.
* `report/`: Chứa file báo cáo chi tiết định dạng PDF và mã nguồn LaTeX.
